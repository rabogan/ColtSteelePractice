Let's Play With Flexbox

display:flex;

The first thing that is done is to enter the container ID within css, and change the display to flex. 

            
                #container {
                    background-color: #003049;
                    width: 90%;
                    height: 500px;
                    margin: 0 auto;
                    border: 5px solid #003049;
                    display:flex;
                }

Flex Direction

Advantages of using flex-direction

  • It allows you to easily control the layout of items within a container in a flexible and responsive way.
  • It allows you to easily align items along a main axis (horizontally or vertically) and a cross axis (perpendicular to the main axis).
  • It simplifies the process of creating layouts that work on different screen sizes and devices.

Disadvantages of using flex-direction

  • It may not work well with older browsers that do not support Flexbox.
  • It is not suitable for creating more complex layouts, such as multi-column layouts or grid systems with irregularly sized items.

Changing the Flex Model

  • In the Flex model, we have a main axis which initially goes from left to right, and a cross axis which intially goes from top to bottom.
  • The flex-direction property determines the direction in which the flex items are laid out within a flex container.
  • By default, flex items are laid out in a row from left to right. However, you can use flex-direction to change the direction to a column from top to bottom, or to rows that are reversed from right to left or bottom to top.
  • To use the flex-direction property, you can add it to your CSS code like so: 
                    
                    #container {
                        display: flex;
                        flex-direction: row; 
                        /* default value */
                        }
  • We can adjust the code in 3 other ways: 
                    
                    #container {
                        display: flex;
                        flex-direction: column; 
                        /* change direction to a column from top to bottom */
                        }
                
            
                    
                    #container {
                        display: flex;
                        flex-direction: row-reverse; 
                        /* change direction to a row from right to left */
                        }
                
            
                    
                    #container {
                        display: flex;
                        flex-direction: column-reverse; 
                        /* change direction to a column from bottom to top */
                        }
