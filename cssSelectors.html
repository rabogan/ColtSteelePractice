Basic Selectors: Universal and Element Selectors

  • First covered: the universal selector (*) is shown, but its use is not recommended. It basically tries to change all elements on the page, so doesn't really meet our needs very often
  • The element selector (e.g. button/h1 etc...) will changes all elements
  • **NEW** The selector list, which uses a format like h1,h2{ ... } allows us to style both h1 and h2

ID Selectors

  • We can use the # id to single in on individual elements (and not just blanket). Before this, I changed the body background-color to #f1faee. The button background-color and font-size were adjusted as well!
  • We wanted ONE specific button, the signup button to have a different color scheme. To do so, I changed the css associated with the id of that signup id
  • One HUGELY important thing: an ID should only be used ONCE on a single HTML page. We shouldn't overuse IDs, but there's no need to avoid using them. A handful on a single page seems to be optimum, but class selectors are much more common

Class Selectors (Lecture 71)

  • The first thing that is changed from the original document, is to change the span. (A reminder that span is a generic inline element). To select ALL spans in the document, use ALT and CLICK on it. ALT+CLICK is for Windows...
  • I named the class '.tag'. It was given a different background-color, font-size, and color
  • Class itself can be applied across a variety of different elements - and is used FAR MORE often than ID selectors

The Descendent Selector

  • For example, we can choose to select all <a>s that are nested inside an <li>. To do so, we can use li a{ ... }
  • So, no li tags would be styled in this...ONLY anchor tags <a> that are inside an <li> tag
  • What happened with this example, is that each section below had a class 'post' appended to it. All anchor tags within that class post were then changed according to the attributes defined within the class
  • Finally, a footer a (generic element selector+element selector descendent selecting) was used in CSS to changed the color of those particular links
